# NOC-APP

# Installation Guide

1. Create `.env` file with the next values

```bash
PORT=3001

MAILER_EMAIL=your-email@example.com
MAILER_SECRET_KEY=your-secret-key

PROD=false
```

> Example in file `.env.template`

2. Install npm modules

```bash
npm install
```

> Also you can use:
> `bash pnpm install`

3. Run with

```bash
npm run dev
```
